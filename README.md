#  Family Bakery Website

 A responsive bakery website template built with HTML, CSS and Bootstrap — perfect for showcasing bakery products and company info beautifully.

🔗 **Live Demo:** https://emanarafa19.github.io/Bakeryproject-Bootstrap/

##  Project Overview

Family Bakery is a modern and visually appealing web page template designed for a bakery business. It includes multiple sections that highlight:

- The bakery's introduction and mission
- Menu and offerings
- Image gallery
- Location & contact Info
- Map integration
- Footer with social links

This template is fully responsive and works on desktops, tablets, and mobile devices.

## 🛠 Technologies Used

| Technology | Purpose |
|------------|---------|
| HTML5      | Page structure |

| Bootstrap 5| Layout & responsiveness |
